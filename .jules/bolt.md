## 2024-05-19 - Zustand Global State Arrays as useEffect Dependencies
**Learning:** Using global state arrays (like cart `items` from Zustand) directly as dependencies in React `useEffect` hooks for data fetching causes redundant API requests. When unrelated properties of the array elements change (e.g., updating the quantity of a cart item), the array reference changes, triggering the `useEffect` even though the core data required for the fetch (e.g., categories or product IDs) hasn't changed.
**Action:** Extract and use primitive derived values (like `categorySlug` strings or joined strings of `productIds`) during the render phase instead of passing the entire array into the dependency array. This ensures the `useEffect` only runs when the actual required data changes, optimizing performance.
